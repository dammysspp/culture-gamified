# Echoes of Elders - Cultural Simulation Prototype

This is a **high-fidelity, fully functional UI/UX web prototype** designed specifically to look like a AAA game platform. It is built to be run locally in your browser and screen-recorded to produce the assets needed for your video presentation pitch.

## How to Use for Video Production

1. **Running the Prototype**: 
   Simply open `index.html` in your web browser (Chrome/Edge/Safari). 
2. **Setup your Recording**: 
   Press `F11` to go full screen. The app automatically constrains itself to a cinematic 16:9 aspect ratio or fits perfectly on a mobile screen if you resize the browser window.
3. **Capture Transitions**:
   Use screen recording software (like OBS Studio or QuickTime) to capture the prototype. The transitions are timed specifically to give video editors clean cutting points:
   - **Enter World Effect**: Go to *Mission Select* -> click *Enter World*. A massive teal portal will consume the screen, leaving a solid color flash. Editors can cut directly from this flash to live-action footage.
   - **Character Zoom In**: On the *Character Selection* screen, click a hero. The card scales up and locks in smoothly before transitioning, perfect for a dynamic camera zoom edit.
   - **Achievement Pop**: In the *HUD* view, press the `[Esc]` debug button to trigger the achievement toast. It features a particle burst and a slide-in effect that you can composite over real-world footage or the game HUD.

## Design System

- **Color Palette**: Deep Indigo (`#0b0c10`), Warm Terracotta (`#e07a5f`), Soft Gold (`#e9c46a`), Teal Highlight (`#45a29e`).
- **Typography**: `Rajdhani` for futuristic, high-tech gaming headers. `Outfit` for highly legible, modern subtext.
- **Visuals**: Incorporates glassmorphism panels, glowing pulse buttons, animated particles, and rich, deep shadows to simulate an Unreal Engine 5 UI overlay.
